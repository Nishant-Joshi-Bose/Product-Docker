* Bootloadable_BlueLED.cyacd  attached with this AN is the cyacd file generated 
* using the project : Bootloadable_Blue , with the following setting:

* Device : CY8C4014LQI-422 - Production Silicon (CY8CKIT-040)
* Software Version : PSoC Creator 3.3 SP2

* After building, the Bootloadable_Blue.cyacd file is generated,then renamed as 
* Bootloadable_BlueLED.cyacd for understandability

* --------------------------------------------------------------------------------*
* Bootloadable_GreenLED.cyacd is generated using the project Bootloadable_Blue 
* with the LED blinking with green color. Assign the Pin_LED to P1[1].
	
* The device configuration and software is same as above.The .cyacd file generated 
* is then renamed as Bootloadable_GreenLED.cyacd for better understandability.
* -------------------------------------------------------------------------------*
* I2C_Bootloader_Red.hex attached with this AN is the bootloader's hex file 
* generated using the project : I2C_Bootloader_Red. 
* The file size may be different between different PSoC Creator build version. 
* And the Bootloadable_GreenLED.cyacd and Bootloadable_BlueLED.cyacd are generated 
* referring to this bootloader hex file.
